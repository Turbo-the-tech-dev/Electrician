Electrician

A mobile app for electricians featuring essential electrical calculators.

About the Project

The Electrician app is a practical mobile application built with React Native + Expo, designed to assist electricians with essential calculations. It provides tools for common electrical laws, making complex calculations quick and accessible on the go.

Current Features

  • **Ohm's Law Calculator:** Calculates Voltage, Current, Resistance, and Power based on any two inputs.
  • **Ampere's Law Calculator:** Provides calculations for straight wires, solenoids, and toroids.
  • Input validation and error handling for all calculators.
  • User-friendly interface suitable for both Android and iOS devices.

The Future of Electrician

The vision for the Electrician app is to become a comprehensive toolkit for electrical professionals. Future enhancements include:

  • **Resistor Color Code Calculator:** A tool to determine resistance values from color bands.
  • **Voltage Divider Calculator:** For analyzing voltage divider circuits.
  • **Capacitance and Inductance Calculators:** Expanding the range of electrical concepts covered.
  • UI/UX improvements for a more polished and intuitive experience, including dark mode.
  • Robust unit and end-to-end testing to ensure reliability and maintainability.

The project's progress and direction will be tracked through the issues in this repository.
